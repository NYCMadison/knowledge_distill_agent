## 三、归属决策操作说明（人工/Agent 按主题决定目标）

1. **输入**：每条产出来自 `蒸馏产出_待归属/` 下的一个 .md 文件，内含**标题**、**类型**（原则/框架/案例/指标/术语）、**主题标签**、正文。
2. **决策步骤**：
   - 读取该条的**标题、类型、主题标签**（必要时扫一眼正文首段或「本质观点」）。
   - 对照**第一节表格**的「适用主题/类型」列，选出**最匹配**的一个目录。
   - 若同时匹配多个目录，优先选**主题最窄、最专用**的（如储能 → about_energy_storage；财务 → about_financials；创业/GTM → about_startup）；否则多数产业/技术/案例可归入 `about_industry_analysis/所有产业链框架汇总/`。
3. **输出**：确定**目标目录**及**目标文件名**（建议 `YYYYMMDD_主题简述_类型.md`）。
4. **执行**：将文件复制/移动到目标目录并命名；从 `蒸馏产出_待归属/` 删除原文件；在本文档下新增「本次迁移记录」节并列出「文件名 → 归属目标」。
